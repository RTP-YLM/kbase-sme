## เงื่อนไข Usage

- นับ "คำถาม" = 1 request ต่อ POST /api/query (cache hit ไม่นับ)
- เอกสาร = นับตามจำนวนไฟล์ที่ active ใน index (ลบแล้วไม่นับ)
- เกิน quota → แจ้ง admin ล่วงหน้า 80% → ซื้อ top-up ได้ (500 บาท/500 ครั้ง)
- ข้อมูลลูกค้า: เก็บบน Supabase (Singapore region) + สำรองข้อมูลรายวัน
- ยกเลิกสัญญา: แจ้งล่วงหน้า 30 วัน คืนข้อมูลในรูปแบบ JSON ภายใน 7 วัน
