# KbaseSME — Lead List & Outreach Framework
*Target: SI / IT Company | Solo founder — เน้น quality ไม่เน้น quantity*

---

## ICP (Ideal Customer Profile) — SI ที่ใช่

**ขนาด:** 10–100 คน  
**Pain ที่ต้องมี:**
- ทีม pre-sales / sales engineer ≥ 2 คน
- มี spec catalog หรือ product catalog (Excel/PDF) ที่ซับซ้อน
- เคยเสียลูกค้าหรือเสีย deal เพราะตอบช้า หรือข้อมูลผิด

**ไม่ใช่ target:**
- SI ขนาดใหญ่ (> 500 คน) — procurement process ยาว ไม่คุ้ม
- IT shop ที่ขายแค่ hardware ไม่มี solution selling
- บริษัทที่ IT team ต้องการ self-host ทุกอย่าง (ยัง)

## Outreach Templates

### Template A — Facebook DM (เพิ่งเจอในกลุ่ม)

```
สวัสดีครับคุณ [ชื่อ]

เห็นโพสต์เรื่อง [หัวข้อที่เขาพูด] ตรงกับสิ่งที่กำลังทำอยู่พอดีครับ

ทำระบบที่ให้ทีม pre-sales ถามคำถามเกี่ยวกับ spec catalog และ TOR ได้ทันที
แทนที่จะต้องเปิด Excel หรือรอคนที่รู้ — ตอบได้ใน 10 วินาที พร้อมอ้างอิงเอกสารจริง

ถ้าสนใจคุยแบบสั้นๆ 15 นาที เผื่อมีประโยชน์ครับ ไม่มีเงื่อนไข
```

### Template B — LINE (มี contact แล้ว)

```
สวัสดีครับ [ชื่อ] 👋

ขอถามตรงๆ เลยนะครับ — ทีม pre-sales ของที่บริษัทใช้เวลาเท่าไรต่อ TOR หนึ่งฉบับ
ในการ match spec กับ product catalog?

กำลังทำ pilot กับ SI 2-3 เจ้า น่าจะตรงกับปัญหาที่เจออยู่
สะดวก video call 15 นาทีสัปดาห์นี้ไหมครับ?
```

### Template C — Follow-up (ส่งไปแล้ว ไม่ reply 5 วัน)

```
[ชื่อ] ครับ ส่ง follow-up มาเผื่อ message หายไปครับ

ถ้าตอนนี้ไม่ใช่ช่วงเวลาที่เหมาะ ไม่เป็นไรเลยครับ
แค่อยากให้รู้ว่า ถ้าวันไหนทีมมีปัญหาเรื่อง spec catalog หรือ TOR matching
ยังติดต่อได้ตลอดครับ

ขอบคุณครับ 🙏
```

## Discovery Call Script (15 นาที)

### เปิด (2 นาที)
> "ขอบคุณที่สละเวลาครับ อยากถามก่อนนะครับ ทีม pre-sales มีกี่คน และปกติรับ TOR มากี่ฉบับต่อเดือนครับ?"

### Discovery (8 นาที)
- "กระบวนการ match spec ปัจจุบันเป็นยังไงครับ?"
- "เคยมีเคสที่ตอบลูกค้าช้าหรือข้อมูลผิดไหมครับ? เกิดอะไรขึ้น?"
- "ถ้ามีระบบที่ตอบได้ทันที — ใครในทีมจะได้ประโยชน์สูงสุด?"

### แสดง value (3 นาที)
> "บริษัทอื่นที่ use case คล้ายกัน — ใช้เวลาจาก 1-2 ชั่วโมงเหลือ 10 วินาทีต่อ TOR
> อยากให้ทีมลองดูไหมครับ? มี pilot 30 วัน ไม่ผูกมัด"

### ปิด (2 นาที)
> "ขั้นต่อไปคือส่งเอกสาร spec catalog มาให้ดู 1-2 ไฟล์ครับ
> จะได้เห็นว่าระบบ handle ข้อมูลของที่นี่ได้แค่ไหน — ส่งได้เลยไหมครับ?"
